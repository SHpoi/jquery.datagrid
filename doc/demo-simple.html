jQuery datagrid demo simple

Options used
url
"data/demo-simple.php" server url. post data : page, paging, orderby and direction
col
[ {field:"fieldname", title:"title", sortable:true}, {...}, ... ] columns definition with title, field name and sortable activated
attr
{"class":"table table-striped table-bordered table-condensed"} table attributes set by $.attr()
sorter
"bootstrap" plugin used to display sort icon
pager
"bootstrap" plugin used to display pagination
